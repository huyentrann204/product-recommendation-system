# 🛍️ Content-Based Product Recommendation System

A group project that develops a **content-based product recommendation system** using product metadata. The system recommends similar products by analyzing textual features with **TF-IDF Vectorization** and **Cosine Similarity**, helping users discover products with similar characteristics.

---

## 📌 Project Overview

This project aims to build a recommendation system that suggests similar products based on their textual information instead of user purchase history. By leveraging Natural Language Processing (NLP) techniques, the system transforms product descriptions into numerical vectors and measures product similarity using cosine similarity.
